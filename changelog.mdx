## v0.12.0 (2026-03-26)

**New features**

- Adjusted the recognition modes to output in two formats: Markdown and LaTeX.

**improvements**

- Updated Mixed Mode so that mixed content (text, formulas, and tables) is output in Markdown format.
- Added Markdown and LaTeX support for the Collection and History buttons.

## v0.11.11 (2026-03-11)

**New features**

- Added an Auto Mode so you no longer need to manually switch recognition modes.
- Added a Mixed Mode to recognize mixed content including text, formulas, and tables.
- Added more providers (Vercel AI Gateway, Gemini, and OpenRouter), allowing you to switch to BYOK mode after the free credits run out.

## v0.11.10 (2026-03-07)

**New features**

- Added a dedicated Copy button for Table Mode.

**Fixes**

- Fixed an issue where History and Collection couldn’t preview mixed text.

**improvements**

- Enabled direct preview of recognized content in Table Mode , without needing to jump to Overleaf.

## v0.11.9 (2026-03-02)

**New features**

- Added a Table Mode to specifically convert images containing tables into LaTeX format.

**Fixes**

- Fixed an issue where the Copy button style was inconsistent with other buttons.

**improvements**

- Upgraded the prompt system.

## v0.11.8 (2026-02-25)

**New features**

- Added Stack and Split screen layout modes.
- Added a Clear button to the Image (OCR) panel to clear the editor, upload, and preview panels with one click.

**Fixes**

- Fixed the Import from URL button not recognizing WEBP images.

**improvements**

- Changed Import from URL from an inline display to a dialog modal.
- Updated the Image (OCR) panel layout to better match the overall app style.

## v0.11.7 (2026-02-09)

**New features**

- Replaced static layout with resizable panels.
- Added LaTeX logo SVG component

**improvements**

- Improved LaTeX logo in preview panel and image in OCR panels responsiveness when resizing panels


## v0.11.6 (2026-01-19)

**New features**

- Added Terms of Service and Privacy Policy pages, with footer links

**Breaking changes**

**Fixes**

- Fixed collection button storage bar color thresholds/background
- Fixed align environment line break handling

**improvements**

- Refactored layout/page container and updated URL import UI (icon/tooltip)
- Improved SVG preview responsiveness
